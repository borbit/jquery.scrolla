jQuery.Scrolla

Simple and lightweight jQuery plugin to make scrollbars for a draggable content. The plugin is hosted on GitHub. Drop me a line to the issues page if you notice a bug. Any contribution is appreciated.

Simple example with one vertical scrollbar for a simple 10 items list. The list is bigger than its viewport only by height, so a horizontal scollbar is absent. Just drag the scrollbar and the list will be scrolled.

Demo #2

The image in this example is bigger than its viewport both by height and width, so there are two scrollbars. The image also can be scrolled only by dragging of scrollbars .

CSS 
#viewport {
    position:relative;
    overflow:hidden;
    height:150px;
}
#viewport img {
    position:absolute;
}
.scrolla-x,
.scrolla-y {
    position:absolute;
    background:#555;
    border-radius:2px;
}
.scrolla-x {
    bottom:0px;
    height:10px;
}
.scrolla-y {
    right:0px;
    width:10px;
}
HTML 
<div id="viewport">
    <img src="/path/to/your/image.png"/>
</div>
JS 
$(function() {
    $('#viewport').scrolla();
});

HTML 
<div id="viewport">
    <ul>
        <li>List item 1</li>
        <li>List item 2</li>

        <li>List item 4</li>

        <li>List item 10</li>
    </ul>
</div>
JS 
$(function() {
    $('#viewport').viewport({position: 'top'});
    
    var content = $('#viewport').viewport('content');
    
    content.draggable({containment: 'parent'});
    content.scraggable({containment: 'parent'});
    
    $('#viewport').scrolla({content: content});
});
