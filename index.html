jQuery.Scrolla

Simple and lightweight jQuery plugin to make scrollbars for a draggable content. The plugin is hosted on GitHub. Drop me a line to the issues page if you notice a bug. Any contribution is appreciated.

Demo

Simple example with one vertical scrollbar for a simple 10 items list. Just drag the scrollbar and the list will be scrolled.

CSS 
#viewport {
    position:relative;
    overflow:hidden;
    height:150px;
    background:#eee;
    border:5px solid #eee;
}
#viewport li {
    padding-left:5px;
    line-height:30px;
}
.scrolla-y {
    position:absolute;
    right:0px;
    width:10px;
    background:#555;
    border-radius:5px;
}
HTML 
<div id="viewport">
    <ul>

JS 
$(function() {
    $('#viewport').scrolla();
});
